# TODO list
A Simple web app for TODO List - that you can run locally from any Browser or install as app.

There is no cloud/service that need to be up and running - its a simple single page App - that helps you to get through the day.
Usefull for every day tasks - instead of writing down notes in Notepad you can use your browser to see a nice looking app
Which allows you to manage your daily tasks.

If you want it locally on your laptop/computer - just download the "todo.html".
And open it up with your favorite browser.
